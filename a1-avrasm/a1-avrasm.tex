 \section{Boolean logic}
    Using Boolean logic,output A in table 0 can be expressed in terms of inputs W,X,Y,Z as
    
    A=Z'W+Z'X+Z'YX'+ZY'X'W'
    
    B=Z'YX'W'+Y'X'W+Z'Y'X
    
    C=Z'X'W+Y'X'W+Z'XW'
    
    D=Z'W+Y'X'W
